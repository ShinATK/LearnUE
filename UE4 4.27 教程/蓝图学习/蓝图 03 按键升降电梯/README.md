# 03-2 按键升降电梯

大体流程和 [蓝图 02 按键鼠标开关门](../蓝图%2002%20按键鼠标开关门/README.md)相同
- 建立蓝图类
- 在根节点下建立碰撞箱和静态网格体
- 在事件图表中设置蓝图

![](img/Pasted%20image%2020240112223535.png)

**值得注意的是**：由于这里想要建立一个垂直升降电梯，所以需要碰撞箱和作为电梯的平台一起移动，*记得将碰撞箱放在平台这个静态网格体层级之下*
