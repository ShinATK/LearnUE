# 对象引用
**提升为变量**，使用该引用，防止图表中拉线过多，并且也可以跨图表使用，避免了反复调用
# 变量有效性
判断当前变量引用是否有效
- 例如触发加速部分，首先判断 overlap 事件，随后对象转换为第三人称角色，进而才调用人物的加速事件
- 为了方便复用，这里用一个变量建立对这个第三人称角色的引用
- 这里的判断变量引用有效性，就是检测这个引用变量是不是空的
